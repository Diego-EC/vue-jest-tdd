# vue-jest-tdd

## Memoria
### Creación proyecto
- Este proyecto está creado con el cli de Vue
  ``` 
  npm install -g @vue/cli
  ```
  ```
  vue create <my-project>
  ```
- Funcionan tanto ```$ jest --watch``` como ```$ npm run test:unit``` para lanzar los test
